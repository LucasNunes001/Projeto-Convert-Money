## 📌 Sobre o projeto

O **Convert Money** é uma aplicação web desenvolvida para realizar conversões de moedas de forma simples, rápida e intuitiva.

O projeto foi criado com foco em praticar conceitos de desenvolvimento front-end, incluindo manipulação de DOM, eventos e formatação de valores.

---

## ✨ Funcionalidades

* 💰 Conversão entre moedas
* 🔄 Atualização dinâmica dos valores
* 🎯 Interface simples e intuitiva
* 📱 Layout responsivo

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
